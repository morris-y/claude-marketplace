# Morris's Claude Code Marketplace

Personal plugin marketplace for Claude Code.

## Available Plugins

| Plugin | Description | Version |
|--------|-------------|---------|
| [blueprint-skills](https://github.com/morris-y/blueprint-skills) | Trading strategy lifecycle skills with postmortem compliance | 1.0.3 |

## Installation

```bash
# Add this marketplace
claude plugins add-marketplace morris-y/claude-marketplace

# Install plugins
claude plugins install blueprint-skills
```

## Adding New Plugins

Edit `marketplace.json` to add new plugins to this marketplace.
